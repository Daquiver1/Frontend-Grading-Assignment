## MISSING GRADE REPORTING SYSTEM
## Technologies Involved
1. React
2. Css

## Pages/Components
1. Homepage
2. Login
3. Dashbboard
4. Grade-Report 
5. Instructor Contact
6. Help and Support
7. Missing Grading Form


## Component Information
1. Homepage:This contains a general overview of the entire system.
2. Login: Users will be required to Login in with their Student ID and Pin.
3. Dashboard: This contains information about the users current grades for the academic year.
4. Grade-Report:This gives the user history if all grades.Users will be able to click buttons to display grades from different academic years.
5. Instructor Contact:This contains the contact information of the lecturers ans also permits you to eamil them.
6. Help and Support: This contains FAQ information and answers to some questions that are mostly asked.
7. Missing Grade Form: This is a form users can fill to resolves any issues with their grades.
